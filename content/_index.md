### Learning Objectives

- Leverage EKS & OpenTelemetry demo app to send logs, metrics, and traces to Logz.io
- Understand the basics of Observability:
  - logs
  - metrics
  - traces
- Use Logz.io to:
  - Search logs
  - Create Dashboards
  - Get Insights into applications via traces
